SIEM LAB Project

Objective

To create a home Security information and event management (SIEM) lab using Elastic and Kali Linux, allowing me to gain hands-on experience in setting up, using, and analyzing security event within a SIEM environment.

Skills Learned

  • Experience Deploying and configuring a SIEM solution utilizing Elastic Stack to monitor and analze security events .
  • Conduceted analysts of security logs, gaining hands on experience identifying anomalies and potential threats
  • OpenVPN configuration.
  • Developed experience in real-time monitoring, gaining practical knowledge of detection, searching and response to incidents.

Tools Used

  • Services: Elastic Stack,Elasticsearch , Elastic Defend.
  • OS: Kali Linux.
  • Terminal: Nmap.
