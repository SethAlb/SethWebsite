Open VPN Project

Objective

The primary objective of this project was to create and deploy a secure and functional VPN on AWS to help me learn and apply network security concepts. The project also included implementing a user-friendly web interface to verify the VPN connection by showing the user their IP and location according to their IP. I needed it to be available 24/7 without downtime to keep this project realistic. I implemented this by using AWSâ€™s Auto Scaling feature, ensuring continuous availability and minimum downtime by ensuring there is always an EC2 running and helping to deliver a cost-efficient solution.
